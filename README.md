# 💰 Sistema de Controle Financeiro

Este é o meu primeiro projeto prático na área de desenvolvimento web, criado com o objetivo de aplicar na prática os conhecimentos adquiridos em Python e Flask.

O sistema permite o controle de receitas e despesas pessoais, com funcionalidades como login seguro, dashboard com gráficos e histórico de transações.

---

## 🔧 Tecnologias utilizadas

- Python (Flask)
- SQLite
- HTML5 e CSS3 (com Bootstrap)
- Jinja2
- Chart.js
- Flask-Mail

---

## 🧩 Funcionalidades

- Login e autenticação de usuários
- Recuperação de senha por e-mail
- Cadastro de receitas e despesas
- Dashboard com gráficos interativos
- Histórico de transações
- Layout responsivo
